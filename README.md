# FETCHTUBE

Ferramenta de terminal feita em Python pra pegar informações de vídeos e (em breve) baixar direto pela URL.

Simples, rápida e com um visual estilizado no terminal.

---

## O que faz

- Baixa vídeos na melhor qualidade disponível  
- Mostra título e autor do vídeo  
- Funciona só com a URL
